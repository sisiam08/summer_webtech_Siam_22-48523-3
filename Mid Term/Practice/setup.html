Method 2: Full Server Mode (Recommended for Development)

For complete PHP functionality:

Option A: Using XAMPP (Windows)

  1. Download and install XAMPP
  2. Start Apache server from XAMPP Control Panel
  3. Copy all login files to C:\xampp\htdocs\login\
  4. Open browser and go to http://localhost/login/login.html

Option B: Using VS Code Live Server Extension

  1. Install "Live Server" extension in VS Code
  2. Right-click on login.html
  3. Select "Open with Live Server"
  4. Note: This only works for frontend demo mode, not full PHP

Option C: Using Python (Simple Server)

  1. Open terminal in the Practice folder
  2. Run: python -m http.server 8000
  3. Open http://localhost:8000/login.html
  4. Note: This only works for frontend demo mode, not full PHP
